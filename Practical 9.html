CSS3 Multiple Columns

CSS3 multiple column layout is used to display content in newspaper style format. It helps divide long content into multiple vertical sections using properties like column-count, column-gap and column-rule. This makes the content look more attractive and readable. It is very useful for articles, blogs and magazines. In responsive web design, the number of columns can be changed according to screen size using media queries. This allows the webpage to look proper on desktop, tablet and mobile devices.
